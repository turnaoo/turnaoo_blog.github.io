AI + Hugo 搭建个人博客#

最近我用 AI 辅助 + Hugo 搭建了这个个人博客，记录一下整个过程和心得。

为什么选择 Hugo？#

Hugo 是一个用 Go 语言编写的静态网站生成器，以速度快、配置简单而著称：

  • 🚀 超快构建速度 - 相比 Jekyll、Hexo 等工具，Hugo 的构建速度堪称惊人
  • 📦 零依赖 - 是一个独立的二进制文件，无需安装额外环境
  • 🎨 丰富的主题 - 社区贡献了大量精美主题
  • 📝 Markdown 支持 - 专注于内容创作，无需关心样式

为什么选择 PaperMod 主题？#

PaperMod 是一个简洁高效的 Hugo 主题，特别适合个人博客：

  • 📱 响应式设计 - 完美适配各种设备
  • 🌙 暗色模式 - 自动跟随系统主题
  • 🔍 内置搜索 - FuseJS 实现的快速搜索
  • 📊 SEO 友好 - 支持 OpenGraph、JSON-LD 等

AI 如何辅助开发？#

在整个搭建过程中，AI 帮了我大忙：

  1. 主题配置 - 帮我理解和配置 PaperMod 的各种参数
  2. 问题排查 - 解决了图片路径、CSS 加载等各种坑
  3. 内容创作 - 辅助写代码片段和技术文档
  4. 部署优化 - 配置 GitHub Actions 实现自动化部署

AI 不是替代者，而是强大的助手。它让我能专注于创意，而把重复性的工作交给它完成。

技术栈#

  • 框架：Hugo + PaperMod 主题
  • 图床：七牛云 / 直接上传
  • 部署：GitHub Pages + GitHub Actions
  • 域名：turnaoo.top

  4. 🚀 不断完善博客功能和内容
